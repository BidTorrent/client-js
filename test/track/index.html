BidTorrent Impression Tracking Test

Uses local configuration files to send bid request to Criteo bidder through local wrapper, expect valid bid, display ad and call local impression tracking endpoint. To run this test you'll need following setup:

  • This demo page hosted behind any web server you want (no restriction on domain or path) ;
  • Criteo bidder wrapper hosted in http://bt-dev.criteo.com/criteoBid.php ;
  • Impression tracker hosted in http://log-dev.bidtorrent.io/imp.php
